# NodeJS

## Windows/Mac
https://nodejs.org/en/  
Download latest "LTS" release  
Install and wait for it to finish  

Check the version to make sure it installed correctly
```
λ node --version
v14.15.3
```

## Linux
```
λ sudo apt-get update
λ sudo apt-get install node
```

## Run
```
λ npm
λ node main.js
```
